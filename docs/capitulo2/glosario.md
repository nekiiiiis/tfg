# Glosario

<div align=center>

|Término|Definición|
|-|-|
|**Usuario**|Operador del sistema perteneciente a Infinite Fieldx.|
|**Token**|Activo listado en Hyperliquid, ya sea spot o perpetuo.|
|**Mercado**|Categoría de un token: **Spot**, **PerpNativo** o **PerpHIP3**.|
|**Spot**|Mercado de tokens spot desplegados en HyperCore mediante HIP-1.|
|**PerpNativo**|Mercado de perpetuos listados oficialmente por Hyperliquid.|
|**PerpHIP3**|Mercado de perpetuos desplegados de forma permissionless por builders mediante HIP-3.|
|**Precio**|Valor en dólares de un token en un instante dado.|
|**Dirección**|Dirección pública de Hyperliquid.|
|**Entidad**|Agrupación de direcciones bajo un nombre asignado por el usuario.|
|**Operación**|Compra o venta ejecutada en Hyperliquid, con volumen en dólares.|
|**Temporalidad**|Período para el leaderboard: 1h, 4h, 6h, 12h, 1d o 1w.|
|**AlertaPrecio**|Alerta que monitoriza el precio de un token respecto a un umbral.|
|**Webhook**|Endpoint URL de destino para notificaciones.|
|**Notificación**|Mensaje enviado al webhook cuando una alerta se dispara.|
|**HIP-1**|Native Token Standard. Estándar de Hyperliquid para desplegar tokens spot nativos en HyperCore.|
|**HIP-2**|Hyperliquidity. Mecanismo de liquidez automatizada para tokens HIP-1.|
|**HIP-3**|Builder-deployed Perpetuals. Estándar que permite el despliegue permissionless de mercados de perpetuos.|

</div>
