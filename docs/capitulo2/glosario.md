# Glosario

Vocabulario común del dominio utilizado a lo largo de todo el capítulo. Los términos se mantienen idénticos en diagramas, casos de uso y prototipos.

<div align=center>

|Término|Definición|
|-|-|
|**Usuario**|Operador del sistema perteneciente a Infinite Fieldx.|
|**Token**|Activo listado en Hyperliquid, ya sea spot o perpetuo.|
|**Mercado**|Categoría a la que pertenece un token. Puede ser **Spot** (HIP-1/HIP-2), **PerpNativo** (perpetuos oficiales de Hyperliquid) o **PerpHIP3** (perpetuos desplegados de forma permissionless).|
|**Precio**|Valor en dólares de un token en un instante dado, obtenido en tiempo real desde Hyperliquid.|
|**Dirección**|Dirección pública de Hyperliquid (cadena hexadecimal que identifica una cuenta).|
|**Entidad**|Agrupación de una o más direcciones bajo un nombre legible asignado por el usuario. Permite identificar operadores en el leaderboard.|
|**Operación**|Compra o venta ejecutada en Hyperliquid (trade), asociada a una dirección y un token, con un volumen expresado en dólares.|
|**Temporalidad**|Período de tiempo seleccionado para consultar el leaderboard: 1h, 4h, 6h, 12h, 1d o 1w.|
|**AlertaPrecio**|Alerta configurada por el usuario que monitoriza el precio de un token respecto a un umbral en dólares.|
|**Webhook**|Endpoint URL externo al que el sistema envía las notificaciones cuando una alerta se dispara.|
|**Notificación**|Mensaje generado automáticamente por el sistema y enviado al webhook cuando se cumple la condición de una alerta.|
|**Leaderboard**|Ranking de las direcciones con mayor volumen de compra o venta de un token, dentro de una temporalidad y un tipo de mercado determinados.|
|**HIP-1 / HIP-2**|Estándares de Hyperliquid para tokens spot nativos y su provisión de liquidez automatizada, respectivamente.|
|**HIP-3**|Estándar de Hyperliquid que permite el despliegue permissionless de mercados de perpetuos por parte de builders.|

</div>
